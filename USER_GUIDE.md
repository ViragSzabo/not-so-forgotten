# User Guide: Not-So-Forgotten

Welcome to your professional archive of memories. This guide will help you navigate the system.

---

## The Home Page
The primary dashboard of the application.
- **System Status**: View the current clock and radio rotation.
- **Record Memory**: Use the input fields to save a new memory to the archive.  
  _Note: Both Title and Description are required!_
- **Quick Navigation**: Access the Memory Archive or Playlist Collection directly.

## The Memory Archive
Browse all stored memories in a professional, minimalist grid.
- **Visual Cards**: Each memory is represented by a clean card with a vertical accent.
- **External Integration**: Access Spotify or YouTube references for any memory directly from its card.

## Music Archive
Manage your curated collections of musical references.
- Create new collections with focused descriptions and Spotify references.
- Delete outdated entries to keep your archive clean.

## Configuration
Customize your experience and manage security.
- **Spotify Client ID**: Essential for music integration.
- **YouTube API Key**: Required for fetching visual media.
- **Save Configuration**: Securely store your keys using platform-level encryption.

---

### Setup Tips
1. **Developer Keys**: To get the full experience, provide your Spotify Client ID and YouTube API Key in the Configuration page.
2. **Premium Feel**: The app is designed for high-contrast, dark environments.
